A modern, responsive, and dark-themed personal portfolio website designed for **Faizal Richi Candra**. This project showcases professional skills, certifications, and experience with a dynamic and interactive user interface.

## ğŸš€ Features

* **Cyber/Tech Aesthetic:** Immersive dark mode design with a moving 3D grid background and glowing effects.
* **Interactive Particle Engine:** Custom HTML5 Canvas implementation for:
    * **Cursor Trail:** Blue energy particles follow the mouse movement.
    * **Click Burst:** Particle explosion effect on mouse click.
    * **Stats Celebration:** Confetti explosion animation when hovering over project statistics cards.
* **Dynamic Animations:**
    * Smooth scroll animations using **AOS** (Animate On Scroll).
    * Typewriter text effect using **Typed.js**.
    * Floating profile image with orbiting tech icons (Code, Gear, Brain, Teaching).
* **Responsive Design:** Fully optimized layout for desktops, tablets, and mobile devices.
* **Smart Certification Gallery:** Displays selected highlights with a "View All" integration directing to LinkedIn for professional verification.
